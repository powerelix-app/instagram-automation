# Контент-стратегия POWERELIX Instagram (v1, 2026-06-10)

Что и о чём постим. Источник принципов: бренд-фундамент + разбор @chatplaceio +
плейбуки (`prompt-lab/playbook-headlines.md`, `playbook-image.md`).

## Цель и тон
- Растить **новый аккаунт с 0**: сначала доверие и охват, потом продажи.
- Тон POWERELIX: **тёплый, честный, семейный**, на «ты». Аутентичность > сухая экспертиза.
- Юр. БАД (РФ): «не лекарство», мягкие формулировки («поддерживает / помогает»), дисклеймер на продуктовых.

## Контент-микс (≈70/20/10)
Польза+Вовлечение (охват/доверие) — Бренд/Лайфстайл — Продукт (продажа). Карусели — приоритет 2026; Reels — охваты.

## 5 рубрик (контент-пиллары)
| # | Рубрика | Цель | Формат | Доля |
|---|---|---|---|---|
| 1 | **Экспертная** — «N сигналов дефицита», зачем нутриент, как выбрать/принимать, мифы о БАД | доверие | карусель | ~35% |
| 2 | **Польза/лайфхаки** — сон, энергия, стресс, иммунитет, питание, сезон | охват | Reels + карусель | ~30% |
| 3 | **Лайфстайл/бренд** — ценности, семья, за кулисами, «почему POWERELIX», личные истории | доверие | Reels/stories/пост | ~15% |
| 4 | **Вовлечение** — опросы, «выбери», чек-листы, вопрос→директ | вовлечение | stories/карусель | ~10% |
| 5 | **Продуктовая** — новинка, состав, «за 30 дней», отзывы/UGC + дисклеймер | продажа | пост/карусель | ~10% |

## Банк тем (боль → продукт)
- Стресс / плохой сон / судороги → **Магний + B6**
- Сердце / мозг / кожа → **Омега-3**
- Детокс / иммунитет / свежесть → **Хлорофилл**
- Кожа / суставы / молодость → **Морской коллаген**
- Женский баланс / цикл → **Инозитол** (и Женский комплекс)
- Аппетит / тяга к сладкому / вес → **Пиколинат хрома**
- Спокойствие / тревога / сон → **Комплекс для нервной системы**
- Энергия / выносливость / спорт → **L-Аргинин**
- Дети / школа / иммунитет → **Детские мультивитамины**
- Мужская энергия / тонус → **Мужской комплекс**

## Форматы и частота
- Старт: **5 публикаций/неделю** (пн–пт). Неделя = 2 карусели + 2 Reels + 1 вовлечение/stories.
- Первые 2 недели — упор на **Экспертную + Польза** (доверие), продажа минимально.
- Цель объёма для роста: **15 Reels + 20 каруселей/мес** (по бенчмаркам).
- Каждый пост по формуле HOOK→CONTEXT→VALUE→PROOF→CTA; первые 2 слайда карусели — решающие.

## Приоритетные продукты (раскачиваем первыми)
**Хлорофилл** и **Инозитол** — топы продаж (по статистике WB) + трендовые темы в соцсетях
(детокс/кожа/энергия и женский гормональный баланс). На них и строим первые продуктовые посты.
Юр.: тему инозитола (гормоны/цикл) подаём мягко — **без заявлений о лечении** (СПКЯ и т.п.) + дисклеймер.

## Запуск аккаунта (батч 3–4 поста сразу, чтобы лента не была пустой)
1. **Пост-«знакомство» / миссия → ЗАКРЕПИТЬ.** Бренд: кто мы, зачем POWERELIX, что тебя здесь ждёт. (Якорь для зашедших, не охватный.)
2. **Карусель / Экспертная — Хлорофилл:** «Зачем все пьют хлорофилл: 5 причин» (детокс/кожа/энергия).
3. **Карусель / Экспертная — Инозитол:** «Инозитол — про женское спокойствие и баланс» (мягко, + дисклеймер).
4. **Reels / Польза (релейтбл, для холодного охвата):** «3 привычки для энергии с утра».
5. **Карусель-вовлечение:** «Каких витаминов тебе не хватает? Мини чек-лист» (вопрос → директ).
