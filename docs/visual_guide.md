# Визуальный гайд POWERELIX Instagram (v1, 2026-06-09)

Выбрана **система 1+3** (на основе тестов на реальной банке):

| Где | Стиль | Описание |
|---|---|---|
| **Лента (посты)** | Светлый минимализм | Кремовые/бежевые фоны, мягкий дневной свет, много воздуха, премиально и доверительно. Чёрная/амбер банка ярко выделяется. Свободная ниша в РФ. |
| **Reels / Stories** | Тёплый лайфстайл | Живые домашние/семейные сцены, тёплый золотой свет, банка в контексте (стол, кухня, руки). Релейтбл, охваты. |
| **Спорт / геройные кадры** | Тёмный премиум | Чёрный фон + акцентный цвет продукта + подсветка. Под спортивную под-линейку (L-Аргинин, мужской комплекс) и «энергию/POWER». |

## Палитра
- Светлый фон: кремовый/бежевый (≈ #F4F1EA … #ECE6DA).
- Лайфстайл: тёплое дерево, домашний textile, зелень растений, золотой час.
- Тёмный: глубокий чёрный + акцентный цвет конкретного продукта (из `data/brand_powerelix.json` → `accent_color`).
- Акцент всегда = цвет продукта (магний — фиолетовый, омега — оранжевый, хлорофилл — мятный и т.д.).

## Свет и композиция
- Свет мягкий, естественный; для лайфстайла — тёплый.
- Банка крупно, по правилу третей, много негативного пространства.
- В лайфстайле — банка в реальном контексте (рука, кухонный стол, прикроватная тумба).

## ⚠️ Производство: «крепкая этикетка»
ИИ-редактор (gpt-image edits) местами искажает мелкий текст этикетки. Поэтому для ФИНАЛЬНЫХ ассетов:
1. Генерируем **фон/сцену без банки** (text-to-image в нужном стиле).
2. **Вырезаем реальную банку** из фото (фон у фото белый → простой вырез) и подставляем на сцену + мягкая тень.
→ Этикетка остаётся идеальной (реальные пиксели), а сцена — брендовой.
Для быстрых черновиков/идей можно gpt-image edits напрямую (но текст проверять).

## Технические форматы
- Пост (лента): 4:5 — 1080×1350.
- Reels / Stories: 9:16 — 1080×1920.
- Карусель: 4:5, единый стиль слайдов.

## Фирстиль (переиспользуем из wb-design / MASTER_BRAND_BRIEF)
Единый бренд на упаковке + WB + Instagram. Источник: `wb-design/output/creative_director_brief/MASTER_BRAND_BRIEF.md`.
- **Мнемоники:** надпись POWERELIX + фигурная «X»; фирменные фигурные скобки `{ }` для плашек (`{ 90 капсул }`, `{ 30 дней }`).
- **Шрифты:** заголовки — **Inter ExtraBold** (или Montserrat Black); текст — Inter/Manrope Regular; слоганы UPPERCASE; курсив запрещён; одно семейство везде. Inter — бесплатный (Google Fonts), берём его как основной.
- **Цвета бренда:** чёрный `#0A0A0A`, белый, серые сабтексты; **акцент = цвет продукта** (таблица hex всех 11 в брифе): хлорофилл `#00C29B`, магний `#8975BF`, хром `#4983CD`, коллаген `#2FCDD7`, инозитол `#FA5098`, нервная `#AC68B3`, мужской `#213D87`, женский `#FC7C9C`, детский `#FFDE00`, L-аргинин `#6FC83B`, омега `#FF8522`.
- **На IG-холсте:** бренд-знаки (X, скобки) на светлом фоне — чёрным или акцентным цветом продукта. Акцент точечно (5–15% площади).
- Запрещено: красный градиент (код PWR), тёмно-синий+фуксия (код QEEP), ядовитые неоны.
- **Логотип-файл:** по брифу передаётся отдельным SVG/AI — в репозитории/ассетах пока НЕ найден. Нужно: получить вектор у пользователя ИЛИ извлечь/воссоздать из рендера банки.
