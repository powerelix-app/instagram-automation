# Карта проекта + 10 причин обосраться (POWERELIX Instagram)

Дата: 2026-06-09. Метод: смысло-кодинг куратора («10 причин» + проход пользовательского пути)
+ 12 промтов Артемия. Принципы куратора: низковисящие фрукты, сначала польза → потом
удешевление, не буксовать, знания в файлах.

## Где мы (карта A–F)
- A. Фундамент бренда — ✅
- B. Анализ конкурентов — ✅
- C. Шапка профиля — ✅
- **D. Инструменты генерации визуала — 🔧 мы здесь, глубоко.** Готово: bottle-карточки
  (3 стиля), hero-оверлей на реальных сценах (block/band/anchor), автогенерация сцен
  (Replicate, фазы 1–2). Не зафиксировано: финальный каркас.
- E. Плейбуки промптов — частично.
- F. **Публикация (постинг/Reels/Stories + расписание) — ❌ не построено.**

Доп.: IG API (профиль/медиа/refresh), генератор контент-плана `generate_plan.py` (ни разу
не запущен).

**Диагноз (по куратору): буксуем на D, аккаунт пустой, публиковать нечем. Низковисящий
фрукт = выпустить первые посты, а не доводить визуал до идеала.**

## 10 причин, почему обосрёмся
1. **Вечная полировка визуала → 0 постов** (буксуем). Фикс: зафиксировать каркас, выпустить 3–5 постов, править по охватам.
2. **Юридика БАД (РФ)** в визуальных надписях/хуках (не только в подписях). Фикс: юр-чек-лист на каждый текст на картинке; дисклеймер где нужно.
3. **AI-сцены на дешёвой flux-dev** — кривизна; для «честный/премиум» яд. Фикс: финалы `--hq`, отбор кадров.
4. **Руки/лица/пальцы у людей** AI ломает. Фикс: ревью перед публикацией, favored кадры без крупных лиц/рук.
5. **Банка-в-сцене = фотожаба** на people-сцене. Фикс: продуктовые подложки (стол/подиум), контактная тень (фаза 3).
6. **Публиковать нечем (F не построено).** Фикс: первые посты вручную ИЛИ строим Content Publishing API.
7. **Токены протухнут молча** (IG 60 дн ~авг 2026, баланс Replicate, ANTHROPIC). Фикс: автопродление IG (cron) + мониторинг баланса.
8. **Стоимость генерации без тормозов** (batch на --hq жжёт). Фикс: дефолт дёшево, лимит batch, лог цены.
9. **Новый аккаунт = 0 охватов.** Идеальные посты в вакуум. Фикс: приоритет Reels/охватным форматам, реалистичные ожидания.
10. **Расфокус/дрейф** — решения по визуалу в чатах, не в методе. Фикс: визуальный метод в файл (раз→два→метод).
11. **Бан/ложная блокировка IG-аккаунта** (новые AI-фильтры Meta, март 2026). Фикс (из разбора @chatplaceio): мы на **официальном Meta API** — за это не банят ✓; дополнительно — включить **2FA**, стабильный IP, один аккаунт на устройство; при ложном бане — селфи-видео + ID, ждать 1–24 ч.

## Проход пользовательского пути (где рвётся)
`идея → контент-план → визуал → подпись → публикация → ответы на комменты`
→ рвётся на **«визуал»** (каркас не выбран) и **«публикация»** (F не построено).

## Рекомендованный путь (низковисящий фрукт первым)
1. Зафиксировать каркас: **block** (посты) + **band** (обложки каруселей); anchor — в архив.
2. Прогнать `generate_plan.py` → 2 недели тем/рубрик/легальных подписей.
3. Выпустить первые 3–5 постов (сначала можно вручную: я генерю сцену+оверлей, ты публикуешь).
4. Только потом: достроить автопостинг (F), оптимизировать визуал, добить фазу 3.
