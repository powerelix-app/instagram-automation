Решено сначала проработать стратегию, потом публиковать. Порядок:
- ✅ **A. Фундамент бренда** — позиционирование, миссия, ценности, тон, слоган «Здоровье. Энергия. Каждый день.» (см. `docs/brand_foundation.md`). Визуальная система выбрана: **1+3** (светлый минимализм лента + тёплый лайфстайл Reels, тёмный — спорт) → `docs/visual_guide.md`.
- ✅ **B. Анализ конкурентов** — qeep/pwr (РФ) + ritual/ghost (US) через Apify → `docs/competitor_analysis_2026-06.md`.
- ✅ **C. Шапка профиля** — слоган + bio + имя (категории) + Taplink; установлена вручную.
- **D. Инструменты генерации** — подход **ГИБРИД**. Картинки: **gpt-image через ProxyAPI** (тем же ключом, что Claude — работает, отдельный сервис пока не нужен). Видео: микс (съёмка + AI, позже). Ключевой приём: реальная банка в AI-сцену; для финала — фон-генерация + вырез банки (чтобы этикетка не искажалась). Реальные фото банок: `~/Downloads/Архив/`.
  - Генератор шаблона поста на Pillow — `ig_automation/post_template.py` (`render_post`): кремовый холст, логотип, Inter ExtraBold заголовок, акцент по цвету продукта, фирменные `{ }`, вырез реальной банки с белого фона (чёткая этикетка). Шрифты Inter/Montserrat уже в системе.
  - **Брендовые надписи-водяные знаки** — `ig_automation/brand_overlay.py` (`render(style,...)`), СВОЯ бренд-ДНК (не копия PWR). Три стиля на пробу: `stamp` (вордмарк+знак, заголовок Montserrat Black, акцент-скобка, слоган-стрип), `chips` (фирменные `{ }` как инфо-чипы вокруг банки), `xmark` (гигантский фирменный знак полупрозрачной подложкой). Фирменный знак — **настоящий** `assets/brand/mark.png` (изолированный, прислал пользователь; белый фон → прозрачность на лету в `brand_mark()`); кодовая версия осталась fallback'ом. Раскладка продукт→банка→акцент(hex)→короткий заголовок/подзаголовок/капсулы — `data/product_assets.json` (все 11 опознаны по контактному листу); `render_product(id, style)` тянет банку из `~/Downloads/Архив`, акцент и реальные капсулы/дни (чипы больше не хардкод). Батч: `output/lineup/` (11×3) + листы сравнения по стилю. TODO: **выбрать финальный стиль** (stamp/chips/xmark); банки-исходники пока в Downloads (не в репо).
- **E. Плейбуки** — визуал + цепляющие заголовки, копим в prompt-lab (по референсам/урокам от пользователя).
- **F. Публикация** — постинг/Reels/Stories по плану + расписание.
