# PLANS — Instagram Automation (POWERELIX)

## ✅ Готово
- Подключён официальный Instagram API (Instagram Login): приложение Meta «POWERELIX Automation», аккаунт @powerelix (BUSINESS), долгоживущий токен 60 дней.
- Все 5 разрешений в dev-режиме: basic, content_publish, manage_comments, manage_messages, manage_insights.
- Каркас проекта: config/.env, обёртка Instagram API (профиль, медиа, refresh токена), линейка продуктов.
- **Генератор контент-плана** через Claude (structured output, юр. правила БАД, баланс рубрик/форматов) — `generate_plan.py`.
- Проверка связи — `check_connection.py`.
