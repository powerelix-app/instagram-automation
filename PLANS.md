# PLANS — Instagram Automation (POWERELIX)

## 🎯 Стратегический этап (перед публикацией) — в работе с 2026-06-08
Решено сначала проработать стратегию, потом публиковать. Порядок:
- **A. Фундамент бренда** — позиционирование, миссия/идея, слоган, визуальный стиль (лого, палитра, шрифты). На сегодня: есть только макеты баночек; фирстиль/лого/слоган — создать.
- **B. Анализ конкурентов (РФ) + аналоги в США** → стратегия (питает A). Instagram не парсится автоматически → анализ по скриншотам/хэндлам + веб-поиск. *Ждём от пользователя 2-3 @конкурента.*
- **C. Шапка профиля** (bio) — из A+B.
- **D. Инструменты генерации** — подход **ГИБРИД**: рутина через API (fal.ai → Flux/Nano-Banana/Kling), hero+Reels вручную (Midjourney/Kling/CapCut). Бюджет: «ради качества». Доступ: смешанно. Видео: микс (съёмка + AI). Ключевой приём: вставлять реальную баночку в AI-сцену (image-edit), не генерировать выдуманную упаковку.
- **E. Плейбуки** — визуал + цепляющие заголовки, копим в prompt-lab (по референсам/урокам от пользователя).
- **F. Публикация** — постинг/Reels/Stories по плану + расписание.
